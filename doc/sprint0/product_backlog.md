# Product Backlog

## Epic 1: Fitness App	
The core of our application. Provide workouts and walk you through exercises.

### Stories
- As a user, I want a timer for exercises and rest periods so I can stay on track during my workouts.
- As a user, I want a button to move to the next set for rep-based exercises so that I can track my progress conveniently.
- As a user, I want embedded reference videos for exercises that I can quickly view during a workout. 
- As a user, I want audio cues for exercise transitions so I can stay focused without looking at my screen.
- As a user, I want to be able to browse exercises and add or remove them to my current workout so that I can make a workout more harder/easier.

## Epic 2: Workout Plan Scheduler
Schedule and track workouts, notify you about workouts, and provide suggestions for future workouts.

### Stories

- As a user, I want to input my available times for a workout so I can schedule my workout times.
- As a user, I want to customize notifications for my workouts so that I can be reminded.
- As a user, I want to choose a workout plan from a provided set of workout plans so that I can start exercising without creating my own plan.
- As a user, I want to personalize my workout by adding or removing exercises from my workout plan so I can workout according to my preference.
- As a user, I want to be able to edit an existing workout schedule, so that I can react to unexpected time commitments.

## Epic 5: Communication and Sharing Accomplishments
Send texts, group chats, planning things out with friends. Show off your badges and medals to other users of the app.

### Stories
- As a user, I want to be able to communicate with other users privately through texts and group chats.
- As a user, I want to be able to share images and videos of my accomplishments so I can celebrate milestones with the community.
- As a user, I want to receive likes and comments on my shared workout achievements so I feel encouraged.
- As a user I want to be able to create a bio page to share details like my preferred workouts, age, and gym playlist so others can connect with me based on similar interests.
